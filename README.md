# Atmos

Atmos é um projeto conceitual de um site de hotel de luxo com navegação cinematográfica baseada em scroll. Ambientado em uma taiga montanhosa no verão, o projeto explora atmosfera, arquitetura e movimento sutil para criar uma experiência digital imersiva e sofisticada.

✨ Conceito

A proposta não é apenas exibir um hotel fictício, mas transmitir luxo silencioso por meio de:

Design minimalista

Tipografia refinada

Movimento suave integrado ao layout


🎬 Experiência

O site é dividido em seções que revelam gradualmente:

Entrada do hotel

Interior

Natureza ao redor

Vista panorâmica final

O movimento funciona como extensão da identidade visual, não como elemento dominante.
